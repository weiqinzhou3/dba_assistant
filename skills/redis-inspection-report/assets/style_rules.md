# Redis Inspection Style Rules

- Use Chinese numbered major headings for DOCX: `一、`, `二、`, `三、`.
- Use numeric minor heading numbers under each chapter.
- Keep table titles concise and topic-specific.
- Split wide tables into smaller tables by topic.
- Apply highlight styling to anomaly terms such as critical, high, fail, OOM,
  down, err, failed, 异常, 失败, 高风险, and 中风险.
- Cluster status should be `正常`, `异常 (...)`, or `证据不足`; do not use `-`
  when state evidence exists.
