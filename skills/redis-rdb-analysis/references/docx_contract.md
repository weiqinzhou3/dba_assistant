# RDB DOCX Contract

When the user requests Word, DOCX, Doc, 文档, 报告, or formal report output, the
agent must call the selected analysis tool with report mode and DOCX format.

If the user supplied `output_path`, pass it exactly. If no output_path is
supplied, omit it and let runtime output policy choose the artifact path.

The final answer must include the generated DOCX artifact path. Do not replace
a requested DOCX with only an inline summary.

The artifact path is generated by runtime/report tooling, not invented by the
model.
